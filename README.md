# AlgoDisplay

## 1.简介
该网站主要目的为在公司内部展示软件算法组的特定算法，例如特定靶标抓取，图像测量，字符识别等算法。

## 2. 功能需求
访问用户可以自主选择算法并且输入特定参数进行演示，在界面可以选择自定义图片上传，并且在上传的图片右边生成经过算法处理后的图片，以展示对比的效果。

## 3. 运行环境
Visual Studio 2022

.NET Core 6.0

Windows 11
